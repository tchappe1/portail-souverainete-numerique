### Quel niveau d'implication souhaitez-vous avoir dans ce chantier ?

Contributeur

### Qu'est-ce qui vous a donné envie de rejoindre ce chantier ?
Dans ce monde d'incertitude il est important pour la Région d'avoir une stratégie claire et opinionnée sur sa souveraineté numérique. Dans une dynamique Belge, voir européenne plus large, nous ne pouvons pas être seulement dépendant de technologies ou géants numériques nord-américains.

### Quand vous entendez le terme « souveraineté numérique », qu'est-ce qu'il évoque pour vous ?
La capacité pour la Région de maîtriser, protéger et contrôler ses propres données, infrastructures et technologies.

### Quels sont selon vous les principaux enjeux de souveraineté numérique pour Paradigm et/ou la Région bruxelloise ?
Les outils utilisés par les administrations : Microsoft, l'IA choisie pour les administrations, le cloud souverain, protection des données, etc.

### Dans votre activité professionnelle, avez-vous déjà rencontré des situations illustrant ces enjeux ?(dépendances technologiques, cybersécurité, données, fournisseurs, cloud, IA, etc.)
Oui, les outils utilisés par Paradigm sont souvent des produits de géants numériques nord-américains : Salesforce pour le CRM, Microsoft pour le workspace et l'IA, Orbeon pour la création de formulaires standards sur le e-guichet IRISbox etc.

### Selon vous, quelles devraient être les priorités du chantier dans les prochains mois ?
Les priorités devraient être de :

- Définir les 2 sujets les plus critiques à faire avancer sur la question de la souveraineté numérique.

Concernant ces 2 sujets, je pense que cela pourrait être :

- Se sortir le plus possible des dépendances technologiques / techno-féodalisme (fournisseurs etc.)
- Adopter une IA européenne souveraine pour la Région
